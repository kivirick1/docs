---
title: "☑️ Checkbox (drawCheckBox)"
---
```lua
drawCheckBox(
	id,
	x,
	y,
	w,
	h,
	text,
	customize
)
```

### Parametreler
- `id` (string) — Checkbox benzersiz kimliği.
- `x, y, w, h` (number) — Konum ve boyut.
- `text` (string) — Etiket metni.
- `customize` (table, optional) — Özelleştirme tablosu.

### Dönüş Değeri
- (boolean) — Checkbox state (`true/false`).

### `customize` Anahtarları
| Anahtar | Tip | Varsayılan | Açıklama |
| --- | --- | --- | --- |
| `radius` | number | `5` | Köşe yuvarlama. |
| `alpha` | number | `1` | Opaklık (0–1). |
| `backgroundColor` | table/color | `{"secondary","dark",255}` | Arka plan rengi. |
| `backgroundHoverColor` | table/color | `{"secondary","hover",255}` | Üzerine gelince arka plan. |
| `lineColor` | table/color | `{"primary","dark",255}` | Çerçeve rengi. |
| `lineHoverColor` | table/color | `{"primary","hover",255}` | Üzerine gelince çerçeve. |
| `textColor` | table/color | `{"text","light",255}` | Metin rengi. |
| `textHoverColor` | table/color | `{"text","hover",255}` | Üzerine gelince metin rengi. |
| `iconColor` | table/color | `{"primary","dark",255}` | İkon rengi. |
| `textFont` | string | `"default"` | Yazı tipi adı. |
| `textSize` | number | `10` | Yazı boyutu. |
| `icon` | string | `''` | İkon (FontAwesome). Sadece seçiliyse çizilir. |
| `iconSize` | number | `12` | İkon boyutu. |
| `line` | boolean | `false` | Çerçeve çizilsin mi? |
| `lineOnHover` | boolean | `false` | Üzerine gelince çerçeve. |

### Notlar
- Hover alanı, sadece kutu değil **kutu + yazı genişliği**: `w + 8 + textWidth`.
- Renk tablosu formatında bu element genelde `{"renk","varyant",255}` şeklinde kullanılıyor (alpha dahil).

### Örnek
```lua
drawCheckBox(
	"remember",
	100,
	150,
	20,
	20,
	"Beni hatırla",
	{
		radius = 4,
		lineOnHover = true,
		iconColor = {"primary","hover",255},
		textColor = {"text","light",255}
	}
)
```
