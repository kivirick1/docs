---
title: "☑️ Checkbox (drawCheckBox)"
---
```lua
drawCheckBox(
	id, 
	x, 
	y, 
	w, 
	h, 
	text, 
	customize
)
```

### Parametreler
- `id` (string) — Checkbox benzersiz kimliği.
- `x, y, w, h` (number) — Konum ve boyut.
- `text` (string) — Etiket metni.
- `customize` (table, optional) — Özelleştirme tablosu.

### `customize` Anahtarları
| Anahtar | Tip | Varsayılan | Açıklama |
| --- | --- | --- | --- |
| `radius` | number | 5 | Köşe yuvarlama. |
| `alpha` | number | 1 | Opaklık (0–1). |
| `backgroundColor` | table/color | `{"secondary","dark"}` | Arka plan rengi. |
| `backgroundHoverColor` | table/color | `{"secondary","hover"}` | Üzerine gelince arka plan. |
| `lineColor` | table/color | `{"primary","dark"}` | Çerçeve rengi. |
| `lineHoverColor` | table/color | `{"primary","hover"}` | Üzerine gelince çerçeve. |
| `textColor` | table/color | `{"text","light"}` | Metin rengi. |
| `textHoverColor` | table/color | `{"text","hover"}` | Üzerine gelince metin. |
| `iconColor` | table/color | `{"primary","dark"}` | İkon rengi. |
| `textFont` | string | "default" | Yazı tipi. |
| `textSize` | number | 10 | Yazı boyutu. |
| `icon` | string | '' | İkon (FontAwesome). |
| `iconSize` | number | 12 | İkon boyutu. |
| `line` | boolean | false | Çerçeve çizilsin mi? |
| `lineOnHover` | boolean | false | Üzerine gelince çerçeve. |

### Örnek
```lua
drawCheckBox(
	"remember", 
	100, 
	150, 
	20, 
	20, 
	"Beni hatırla", {
    	radius = 4,
    	lineOnHover = true,
    	iconColor = {"primary","hover"},
    	textColor = {"text","light"}
	}
)
```
